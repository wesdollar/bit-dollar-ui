---
description: "Adds space between elements."
labels: ["ui", "utility"]
---

import { Space } from "./space";

Space can set space by either width or height. Space defaults to a height of `"20px"` and width of `"0"`.

```js
<MyComponent />
<Space />
<ComponentTwo />
```

```js live
<>
  <div>Hello</div>
  <Space height="40px" />
  <div>World</div>
</>
```
